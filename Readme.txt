/////////////////////////////Part 1///////////////////////////////////////////////////////////////////////////////

1) Part1.py uses google api to search results for related keyword 
2) The file returns a file named Search_Data.json and Search_Data_Links.json 
3) Both JSON files contains relevant information about keyword returned from google search using api
4) Enter Api key and Search EngineID and define num and start param if want result after 10
5) After providing necessary information Run the program the links will be returned in Search_Data_Links.json file
